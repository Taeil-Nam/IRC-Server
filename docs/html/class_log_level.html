GlobalLogger 클래스에 대한 별칭을 제공하는 클래스

이 클래스는 GlobalLogger의 기능을 그대로 상속받으면서
로그 레벨을 좀 더 명시적으로 표현하기 위해 사용돤다.
주로 로그 레벨의 가독성을 높인다.
이 클래스는 GlobalLogger의 모든 기능을 포함하며, 추가적인 기능은 제공하지 않는다.
