로그 메세지를 기록

입력된 메세지와 함꼐 로그를 기록한다.
로그 레벨과 발생위치, 호스트이름도 함께 기록 된다.

매개변수
level로그의 심각도 레벨
message로그로 기록할 메세지
functionName로그가 발생한 함수의 이름 (Debug 레벨만)
fileName로그가 발생한 파일의 이름 (Debug 레벨만)
lineNumber로그가 발생한 라인 번호 (Debug 레벨만)

void GlobalLogger::SetTarget ( int  fd)

로그 메세지의 출력 대상을 fd로 지정

로그 메시지를 출력할 파일 디스크립터를 설정한다.
이 설정을 통해 로그 메시지를 특정 파일이나 장치에 직접 출력할 수 있다. 시스템 콜을 이용하기 때문에 속도가 느리다.

◆ SetLevel()

void GlobalLogger::SetLevel ( eSeverityLevel  level)

출력할 로그의 레벨 제한을 지정한다.

이 메소드를 통해 기록할 최소 메크로 레벨을 지정할 수 있다.
예를 들어 Error 레벨로 지정한다면 Error 레벨 이하의
Informational과 같은 로그는 출력되지 않는다. 기본값은 Informational로 설정됨.(따로 설정하지 않을 시)

매개변수
level제한할 로그 레벨
