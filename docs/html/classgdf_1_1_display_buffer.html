상세한 설명

디스플레이에 표시되어야하는 텍스트를 저장하는 클래스

텍스트 콘텐츠를 색상, 시간과 함께 저장하고, 버퍼에 저장된 컨텐츠를 관리한다. 헤더와 푸터를 설정할 수 있으며, 버퍼 용량도 설정 가능하다.
